# ReactJS News Challenge

## Briefing
Good day ladies and gentle men, my name is Chris(a random angmo) and I am looking for the best developers in Singapore. I have just had a chat with your instructional team and they have assured me I am in good hands with your team. I want to be sure you are who your instructors say you are so I have this full day task to test your skills, design and logic skills.

Your task is to develop a news website using the news api where users can read daily news. Your target audiences are people in Africa, Asia, Europe and of course America.

1. The site home should display headline news
1. The site to display on the home page news based on categories.i.e. sports, covid news, politics...
1. The user should be able to navigate and read news from these 4 continents from the navigation bar.


## Get Started
1. Fork and clone this repository
1. Follow instructions below

## Instructions
1. Create a new react app `npx create-react-app newagency`
1. Add Bootstrap if needed 
1. You can install all (bootstrap, axios, react-bootstrap )in one line `npm install bootstrap react-bootstrap axios`
1. Register an account on [News Api Site](http://newsapi.org/) to get api key
1. Get coding.
